下载 汉化包：[Github Release](https://github.com/zetaloop/PowerToys-CN/releases/latest) | [蓝奏云 密码zeta](https://zeta.lanzouq.com/b01lvlllc)

## 使用方法

① 安装原版 PowerToys 软件：

　　下载 .exe 安装包，安装 PowerToys 软件

② 安装优化补丁包：

　　下载 PowerToysCN-汉化版本号.7z，解压后运行“安装.CMD”，根据提示确定安装

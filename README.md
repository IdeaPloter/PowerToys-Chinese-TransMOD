## Usage 使用方法

① 安装原版 PowerToys 软件：

　　下载 .exe 安装包，安装 PowerToys 软件

② 一键安装优化补丁包：

　　下载 PCTMODx***.7z（带有自动安装程序），解压后运行“安装.CMD”，根据提示确定安装

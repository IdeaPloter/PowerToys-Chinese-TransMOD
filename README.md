## Usage 使用方法

① 安装原版 PowerToys 软件：

　　下载 .exe 安装包，安装 PowerToys v0.36.0

② 一键安装优化补丁包：

　　下载 PCTMODx***.7z（带有自动安装程序），解压后运行“安装.CMD”，根据提示完成安装

需要系统版本 Windows 10 1903 或更高，否则可能安装失败。
